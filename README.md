# AWS SERVERLESS API

A simple solution to create a serverless python FastAPI application.
Dependencies are bundles in an AWS Lambda Layer to keep the Lambda Function itself small.


## Technologies

- python
- fastapi
- mangum
- terraform

## AWS Services

- AWS API Gateway
- AWS Lambda
- AWS Lambda Layer
